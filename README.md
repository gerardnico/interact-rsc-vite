# Interact React Rsc Implementation

This is an interact publication framework implementation
based on [Vite React Rsc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-rsc/README.md)

## Note

* The main entry that receive an HTTP request is the `handler` function
  at [entry.rsc](src/interact/rsc/server/entry.rsc.tsx)
* The static build is done with [vite-rsc-ssg](src/interact/rsc/static-generation)
