---
title: How to create a programmatic page (Jsx, Tsx, Js)?
---

A programmatic page is a [page](page.md) created in one of the following language: `Jsx`, `Tsx`, `Js`

## Example

```jsx
// pages/my-page.jsx
export const frontmatter = {
    layout: "landing"
}

export default function () {
    return (
        <p>You can create a `jsx` or `tsx` page</p>
    )
}
```

## Typescript

```typescript
// pages/my-page.tsx
import type {InteractFrontmatter} from "@combostrap/interact/types";

export const frontmatter: InteractFrontmatter = {
    layout: "landing"
}
```
