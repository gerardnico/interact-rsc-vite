Content component are [components](component.md#type) that can be used:

* in [mdx pages](mdx.md)
* in [md pages](md-page.md) if the chosen format is `mdr` or `mdx`

## How to create your own content component

See [How to add a Markdown component](../howto/add-a-content-component.md)

## Default

To see the default content components, check the [markdown syntax page](markdown-syntax.md)
