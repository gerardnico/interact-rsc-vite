---
title: Vite
---

`Interact` is built on top of the [Vite bundler](https://vite.dev/config/)

If you want to add or change its configuration, you can add
a [vite.config.(ts|js)](https://vite.dev/config/) file
in the [root directory](directory-layout.md).

If you start the [cli](cli.md) with the `info` loglevel, you will also get  
the resolved configuration in the file `vite-resolved-config.json` located in
the [build directory](directory-layout.md).
