title: Interactive Components
---

Interactive components are [components](component.md) where the users can
interact with it.

## use client directive

Because they use the Browser (Browser API), they need to have the [use client directive](rsc.md#use-client)
at the top of the file.

## Base UI Library

Our own components add interactivity thanks to [Base-UI](https://base-ui.com/), an unstyled UI components.

They made it easy to have an interactive flow such as opening and closing window that
will always been well positioned (ie on screen)

Base UI is:

* supported by [shadcn](shadcn.md)
* the next evolution of Radix, Floating UI, and Material UI
